🚀 Meus Primeiros Exercícios em Flowgorithm

Olá! 👋 Bem-vindo(a) ao meu repositório de atividades de lógica de programação. Aqui estão alguns exercícios desenvolvidos utilizando o Flowgorithm, uma ferramenta que ajuda a entender programação por meio de fluxogramas.
📚 Atividades Realizadas

Neste material, apresento a resolução de alguns exercícios de lógica, que envolvem entrada de dados, cálculos simples e estruturas condicionais. Esses desafios ajudam a desenvolver o raciocínio lógico e a compreender melhor como funcionam os algoritmos na programação.

Exercício 1: Cumprimento com Nome!
🎯 Objetivo da Atividade:
Praticar a entrada e saída de dados e entender como usar uma variável para armazenar o nome do usuário.
Lógica: O programa pede o nome do usuário, guarda na variável nome e depois exibe a mensagem “Olá” junto com o nome digitado. É um exemplo simples de interação com o usuário.

Exercício 2:Soma de Números!
🎯 Objetivo da Atividade
Praticar a entrada de dados e realizar uma operação de soma entre dois números.
💬 Comentário sobre a Lógica
O programa declara duas variáveis (x e y), pede para o usuário digitar os dois valores e depois calcula a soma entre eles (x + y).
Em seguida, o resultado é exibido na tela. É um exercício básico para entender como receber dados, fazer cálculos e mostrar resultados em um algoritmo.

Exercício 3: Maior ou Menor de Idade!
🎯 Objetivo da Atividade
Praticar o uso de estruturas condicionais para verificar se uma pessoa é maior ou menor de idade.

💬 Comentário sobre a Lógica
O programa pede a idade do usuário e verifica se idade ≥ 18. Se for verdadeiro, mostra que a pessoa é maior de idade; caso contrário, informa que é menor de idade.

Exercício 4:Identificação de Algarismos!
🎯 Objetivo da Atividade
Praticar o uso de condições para identificar se um número é positivo ou negativo.

💬 Comentário sobre a Lógica
O programa pede um número ao usuário e verifica se numero < 0. Se for verdadeiro, mostra que o número é negativo; caso contrário, indica que é positivo.

Exercício 5: Análise de Média do Aluno!
🎯 Objetivo da Atividade
Praticar o cálculo da média de notas e utilizar estruturas condicionais para definir a situação do aluno.

💬 Comentário sobre a Lógica
O programa pede duas notas, calcula a média e verifica o resultado. Se a média for maior ou igual a 7, o aluno é aprovado; se for entre 6 e 7, fica de recuperação; e se for menor que 6, está reprovado.
